# cube
Small project implementing an animated rubiks cube built on top of the first part of the learn opengl tutorial (https://learnopengl.com/).

Rotations are user-controlled via specific keys. Rotations include: front, front inverted, back, back inverted, left, left inverted, right, right inverted, top, top inverted, down, down inverted.
